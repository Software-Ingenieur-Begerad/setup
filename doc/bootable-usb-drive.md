# Create Bootable USB Drive

* download debian

* mount USB drive

* identify USB drive using
```
df -h
```

* unmount drive using
```
sudo umount /dev/<tbd>
```

* format drive using
```
sudo mkfs.vfat /dev/<tbd>
```
